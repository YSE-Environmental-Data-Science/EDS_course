Additional Resources: Coding

This page is a guide for choosing the right resource when you get stuck. The list is intentionally curated rather than exhaustive. Start with the resource that matches your problem, then go deeper if you need more detail.

Git and GitHub

  • Collaborative Coding with GitHub: A beginner-friendly workshop on using Git with GitHub and RStudio. This is especially useful for group work because it emphasizes workflows, conflicts, branches, issues, and project documentation.

  • Happy Git and GitHub for the useR: A comprehensive guide to Git, GitHub, and RStudio. Use the troubleshooting page when authentication, pushing, pulling, or remotes get weird.

  • NCEAS Introduction to Git and GitHub and Collaborating with Git and GitHub: Clear lesson-style materials for learning Git commands and collaboration patterns.

Tidyverse and data wrangling

  • R for Data Science: The best general-purpose reference for tidy data workflows. Particularly useful sections include data visualization, data transformation, joins, tidying, functions, and Quarto.

  • Coding in the Tidyverse: A workshop covering dplyr, tidyr, and ggplot2 for wrangling, summarizing, reshaping, joining, and visualizing data.

  • Posit Cheatsheets: Quick references for common package syntax. Keep the dplyr, tidyr, and ggplot2 cheatsheets close while working through the course.

  • NCEAS Cleaning and Wrangling Data: A lesson on tidy data and wrangling that pairs well with Data Basics and Data Integration.

Spatial data

  • Introduction to Geospatial Concepts: A conceptual introduction to raster data, vector data, coordinate reference systems, spatial resolution, and extent.

  • Introduction to Geospatial Raster and Vector Data with R: A practical workshop for manipulating and plotting raster and vector data with terra, sf, and ggplot2.

  • sf reference site: Function documentation and examples for vector data. Use this when working with points, polygons, joins, intersections, buffers, and distances.

  • Spatial manipulation with sf cheatsheet: A quick reminder sheet for common sf operations.

  • terra reference site and Spatial Data with terra: Documentation and tutorials for raster workflows such as cropping, masking, projecting, resampling, extracting, stacking, and predicting.

  • NCEAS Working with Spatial Data: A lesson using sf, dplyr, ggplot2, and leaflet to manipulate and visualize vector data.

Reporting and reproducibility

  • Quarto Guide: The main reference for writing reproducible reports, websites, slides, and manuscripts with code and narrative text together.

  • Quarto Markdown Basics: Use this when formatting headings, lists, links, figures, tables, and code blocks.

  • Quarto Computations: Using R: Use this when controlling code output, messages, warnings, figures, and execution options in .qmd reports.

  • R4DS Quarto chapter: A friendly introduction to using Quarto for reproducible data analysis.

Modeling and interpretation

  • randomForest package manual: Reference for randomForest(), predict(), importance(), and varImpPlot(). Use this when checking arguments such as ntree, mtry, and importance.

  • R4DS model basics: A gentle introduction to model thinking in R. It does not replace the course Random Forest workshop, but it is useful for thinking about predictors, response variables, and model diagnostics.

  • Tidymodels Learn: A broader modeling resource. This course uses randomForest directly, but tidymodels materials are useful for understanding resampling, recipes, model comparison, and evaluation metrics.

Environmental data products

  • AmeriFlux data variables: Use this when interpreting flux variable names, timestamps, gap-filled variables, and quality-control conventions.

  • FLUXNET2015 dataset: Background on the global flux dataset used in many ecosystem modeling studies.

  • Daymet overview: Background on gridded daily weather data for North America. Useful for climate summaries around Yale-Myers Forest and other point locations.

  • daymetr package: R package for downloading Daymet data directly into R.

  • ERA5 data documentation: Reference for ERA5 climate and reanalysis variables used in Data Integration and Spatial Model Visualization.

  • WAD2M paper: Background on the Wetland Area and Dynamics for Methane Modeling product used to scale spatial methane flux predictions by inundated area.

How to use resources well

When you get stuck, write down three things before searching:

  1. What object are you working with? For example, data.frame, sf, SpatRaster, model object, or Quarto document.
  2. What operation are you trying to do? For example, join, summarize, crop, project, predict, render, or push.
  3. What error message or unexpected result did you get?

Then search using the package name and function name together, such as terra resample, sf st_join, dplyr left_join, randomForest importance, or Quarto code chunk options.
