Raster Data: Climate Exposure and Tower Similarity

Mapping climate surfaces

Vector data told us where FLUXNET CH4 towers are located. Raster data lets us ask what environmental conditions surround those towers. In this workshop, we will use gridded climate layers to evaluate the climate exposure represented by methane tower sites and to identify towers that may be useful analogs for thinking about methane fluxes in Yale-Myers Forest wetlands.

We will ask:

What climate conditions do FLUXNET CH4 towers represent, and which towers are most similar to Yale-Myers Forest?

By the end of this workshop, you will be able to:

  1. Read and inspect raster data with terra.
  2. Understand SpatRaster and SpatVector objects.
  3. Crop and mask rasters to an area of interest.
  4. Use raster algebra to create new climate layers.
  5. Summarize raster values globally and by polygon zones.
  6. Extract raster values to point locations.
  7. Compare tower-site climate exposure to gridded climate space.
  8. Calculate environmental dissimilarity among tower sites.
  9. Use multidimensional scaling (MDS) to visualize which towers are more or less similar.

What is a raster?

A raster is a spatial data structure that subdivides an area into cells, often called pixels. Each cell stores one or more values. Climate rasters are useful because they let us describe environmental gradients everywhere, not only at point locations.

The terra package is built around two important classes:

  • SpatRaster: gridded raster data.
  • SpatVector: vector data such as points, lines, or polygons.

Read climate rasters

This course includes a global climate raster with four layers:

  • MAP: mean annual precipitation in mm.
  • TMIN: mean minimum temperature in degrees C.
  • TMAX: mean maximum temperature in degrees C.
  • MAT: mean annual temperature in degrees C.

Raster algebra

Raster algebra lets us create new layers from existing layers. Temperature range is a simple measure of climate exposure:

temp_range <- global_climate$TMAX - global_climate$TMIN
names(temp_range) <- "temp_range"

Combine the original climate layers with the new temperature range layer:

climate_stack <- c(global_climate, temp_range)
names(climate_stack)
[1] "MAP"        "TMIN"       "TMAX"       "MAT"        "temp_range"

Plot the new raster:

plot(climate_stack$temp_range,
     main = "Temperature range")

Use global() when you want one summary value for an entire raster layer:

global(climate_stack, fun = "mean", na.rm = TRUE)

Crop and mask to an area of interest

Cropping trims a raster to a rectangular extent. Masking removes cells outside an area of interest. Here we use countries from rnaturalearth to create a South America polygon and summarize climate over that region.

world <- ne_countries(scale = "medium", returnclass = "sf") %>%
  filter(continent %in% c("Europe", "Asia", "North America",
                          "South America", "Africa", "Oceania")) %>%
  st_make_valid()

south_america <- world %>%
  filter(continent == "South America") %>%
  st_union() %>%
  st_as_sf()

Crop and mask mean annual precipitation to South America:

south_america_ppt <- climate_stack$MAP %>%
  crop(south_america) %>%
  mask(south_america)

     main = "Mean annual precipitation in South America")

Zonal summaries

Zonal summaries calculate raster statistics inside polygon zones. Here we calculate mean annual precipitation for each country in South America. Keeping the example regional makes the computation faster while preserving the same workflow.

south_america_countries <- world %>%
  filter(continent == "South America")

country_ppt <- zonal(x = south_america_ppt,
                     z = vect(south_america_countries),
                     fun = "mean",
                     as.polygons = TRUE,
                     na.rm = TRUE)

What did zonal() return?

class(country_ppt)
[1] "SpatVector"

Convert the result back to an sf object and map it:

country_ppt_sf <- st_as_sf(country_ppt)

  scale_fill_viridis_c(na.value = "transparent") +

Environmental dissimilarity among CH4 towers

To identify which towers are more or less similar to Yale-Myers Forest, we can compare sites using their raster-derived climate variables. We will use standardized Euclidean distance so variables with large units, such as precipitation, do not dominate the comparison only because of scale.

For this first-pass analysis, we will compare sites using:

  • raster_MAP;
  • raster_MAT;
  • raster_TMIN;
  • raster_TMAX;
  • raster_temp_range.
climate_vars <- c("raster_MAP", "raster_MAT", "raster_TMIN",
                  "raster_TMAX", "raster_temp_range")

analog_candidates <- climate_context %>%
  filter(site_group %in% c("CH4 site", "Yale-Myers reference")) %>%
  select(SITE_ID, SITE_NAME, IGBP, site_group, all_of(climate_vars)) %>%
  drop_na()

Scale the variables and calculate distances:

scaled_climate <- analog_candidates %>%
  select(all_of(climate_vars)) %>%
  scale()

climate_distance <- dist(scaled_climate)

Calculate each CH4 tower’s distance from Yale-Myers Forest:

yale_index <- which(analog_candidates$SITE_ID == "Yale-Myers")

distance_to_yale <- as.matrix(climate_distance)[yale_index, ]

yale_myers_analogs <- analog_candidates %>%
  mutate(distance_to_yale = as.numeric(distance_to_yale)) %>%
  filter(SITE_ID != "Yale-Myers") %>%
  arrange(distance_to_yale)

Interpret analogs carefully

This is a climate-similarity analysis, not proof that a tower is a perfect analog for Yale-Myers wetlands. Methane fluxes also depend on water table depth, wetland type, soils, vegetation, disturbance history, and measurement design. The goal is to identify towers that may be useful starting points for comparison.

MDS visualization

Multidimensional scaling (MDS) takes a distance matrix and places sites into a low-dimensional space. Sites that plot closer together are more similar based on the variables used to calculate the distance.

mds <- cmdscale(climate_distance, k = 2)

mds_scores <- analog_candidates %>%
  bind_cols(as.data.frame(mds) %>%
              rename(MDS1 = V1, MDS2 = V2))

Label Yale-Myers and the five most similar CH4 towers:

label_sites <- mds_scores %>%
  filter(SITE_ID == "Yale-Myers" |
           SITE_ID %in% yale_myers_analogs$SITE_ID[1:5])

Plot the MDS:

ggplot(data = mds_scores,
       aes(x = MDS1, y = MDS2)) +
  geom_point(aes(color = site_group, shape = IGBP == "WET"),
             alpha = 0.75,
             size = 2.5) +
  geom_text(data = label_sites,
            aes(label = SITE_ID),
            nudge_y = 0.12,
            size = 3) +
  labs(title = "MDS of CH4 tower climate similarity",
       subtitle = "Distance calculated from raster-derived climate variables",
       x = "MDS axis 1",
       y = "MDS axis 2",
       color = "Site group",
       shape = "Wetland tower")
